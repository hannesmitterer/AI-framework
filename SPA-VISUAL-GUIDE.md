- On tablet: 2-column grid
- On desktop: 4-column grid for metrics, 2-column for content

## Animations
- Cards fade in as they scroll into view
- Metrics indicators pulse continuously
- Hover effects on all interactive elements
- Smooth transitions on filter changes
