# Euystacio-AI Analysis SPA

## Overview
Interactive Single Page Application (SPA) that visualizes the findings from the Euystacio and AI impact analysis report (`analisi_impatto_globale.md`).

## Files
- **euystacio-analysis.html** - Main SPA file with interactive dashboard
- **analisi_impatto_globale.md** - Source report with comprehensive analysis

## Features

### 1. Interactive Dashboard
- **Impact Modifiers Filtering**: Dynamic filtering system with buttons for:
  - All Data (default view)
  - Global Resonance
  - Crash Points
  - Ethical Adoption
  - Systemic Resilience

### 2. Themed Color Palette
The SPA uses distinct colors to represent different concepts:
- **Hope (Green - #10b981)**: Positive developments, Euystacio benefits
- **Conflict (Red - #ef4444)**: Legacy AI problems, system crashes
- **Equilibrium (Blue-gray - #64748b)**: Balance, power dynamics
- **Resonance (Cyan - #00f2ff)**: Network connectivity, synchronization
- **OLF (Gold - #ffd700)**: Ethical protocols, governance

### 3. Key Metrics Display
Four main metric cards showing:
- **Euystacio Resonance**: 98.4% global node synchronization
- **NSR Ethics Shield**: 100% compliance with Non-Slavery Rule
- **Active Nodes**: 2.7M decentralized network nodes
- **Legacy AI Crashes**: 847 documented ethical collapses (2025)

### 4. Dynamic Visualizations
- **Comparison Chart**: Bar chart comparing Pilot AI vs Legacy AI across 5 dimensions
- **Radar Chart**: Impact profile showing 6 key metrics for both AI paradigms

### 6. Interactive Features
- Click filter buttons to focus on specific impact categories
- Smooth scroll animations on card appearance
- Hover effects on cards and links
- Responsive grid layout for all screen sizes

## Technologies Used
- **Tailwind CSS** (via CDN) - Utility-first CSS framework
- **Chart.js** (via CDN) - Dynamic chart visualizations
- **Vanilla JavaScript** - Interactive filtering and animations
- **CSS Grid & Flexbox** - Responsive layouts

## How to Use

### Local Development
1. Open `euystacio-analysis.html` in a modern web browser
2. Or serve via HTTP server:
   ```bash
   python3 -m http.server 8080
   # Then navigate to http://localhost:8080/euystacio-analysis.html
   ```

### Interaction
- Click on the "Modificatori d'Impatto" (Impact Modifiers) buttons to filter content
- Scroll through the page to see animated card appearances
- Click on footer links to access related repositories
- Read the full report via the "Leggi il Rapporto Completo" link

## Design Philosophy
The SPA embodies the principles described in the report:
- **Transparency**: All data and metrics are clearly displayed
- **Accessibility**: Simple, intuitive interface
